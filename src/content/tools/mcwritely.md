McWritely is a native macOS tool for translation and text improvement that works directly inside any app. There is no separate window to paste into and no copy-pasting back and forth. You write where you normally write, and McWritely helps from there.

Translation and proofreading happen in a single workflow, so you can clean up and convert text in one pass rather than juggling two different tools.

It stays out of your way. There are no distracting overlays and no squiggly underlines covering your screen, which keeps the writing surface clean while you work.

The customization is where it gets interesting. McWritely is fully configurable through system prompts, so you can shape exactly how it edits and translates to match your voice and preferences.

One thing to be clear on: McWritely is not built to generate or rewrite text from scratch. You write something yourself first, and then it helps you make it right. If you want an AI to draft for you from a blank page, this is not that tool. If you want your own writing polished and translated without friction, it fits well.

It is lightweight and affordable, with plans starting at 4 dollars a month. New users can use the promo code "Smart" for 33% off the first month.
